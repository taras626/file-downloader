Техническоое задание:

Реализовать программу для скачивания файлов по url ссылке. Программа должна поддерживать задание 1 файла или списка файлов для скачивания.
Должен отображаться прогресс в процентах для каждого файла, исходный и скаченный размер файла. Должна быть возможность приостановки загрузки и ее удаления из списка.

Список задач:
-Реализовать графическое окно
-Реализовать меню
-Создать функцию добавления файла для скачивания, 
-Создать функцию приостановки скачивания,
-Функция продолжения скачивания с того же места при закрытии(перезапуске) программы
-Функция отображения прогресса скачивания
-Функция отображения всего списка скачивающихся(скачавшихся) файлов
-Функцияя удаления из списка скачивания
